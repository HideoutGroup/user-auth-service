2、支持web应用、rest api

3、PC端支持微信扫码、QQ一键登录、支付宝、手机验证码、用户名与密码

4、移动端支持微信公众号、手机验证码、用户名与密码

6、密码模式输入错误一次后需要输入验证码，错误5次将冻结

7、用户冻结、解锁、踢下线、状态查询
